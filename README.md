# 网咖计时系统

> 桌面应用

## 技术栈

* Electron
* react
* ECMAScript 6
* webpack
* Less

# 开发

首次安装依赖
```
ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron yarn install
```

安装依赖
```
yarn install
```

webpack 打包监听文件改动
```
yarn build
```

启动应用
```
yarn start
```

## 功能

- 下线提醒
- 下线数据变色提醒
- 目前在线人数统计
- 即将下线人数统计
- 更改金额、删除、更新
- 倒计时
- 更改数据的时候弹窗确认
- 数据5分钟保存到特定位置
