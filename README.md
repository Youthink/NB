# 网咖计时系统

> 桌面应用

## 技术栈

* Electron
* react
* ECMAScript 6
* webpack
* Less

# 开发

首次安装依赖
```
ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron npm install

or

ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron yarn install
```

安装依赖
```
npm install

or

yarn install
```

开发
```
npm dev

or

yarn dev
```

启动应用
```
npm start

or

yarn start
```
