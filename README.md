# 网咖计时系统

> 桌面应用

## 技术栈

* Electron
* react
* ECMAScript 6
* webpack
* Less

# 开发

首次安装依赖
```
ELECTRON_MIRROR=https://npm.taobao.org/mirrors/electron yarn install
```

安装依赖
```
yarn install
```

webpack 打包监听文件改动
```
yarn build
```

启动应用
```
yarn start
```

## 功能

- 更改金额、删除
- 删除数据的时候弹窗确认
- 机器编号存在且倒计时没结束，同样的编号不能进入
